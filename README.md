# Setup
```bash
# Clone this repository 
- git clone https://github.com/pedrolaraburu/learning_flutter
# Open your terminal
- Type `flutter pub get`
- Type `flutter run`
```

# Run Unit tests 
```bash
# Open your terminal
- Type 'flutter test'
```

# Run Integration tests 
```bash
# Open your terminal
- Type 'flutter test .\integration_test\app_test.dart'
```
